🔍 SCRIPT TYPES EXPLAINED
==========================

[SERVER SCRIPT]
- Runs on the server only
- Has access to Players and game logic
- Cannot access LocalPlayer or mouse
- Location: ServerScriptService
- Example: PlayerManager.lua

[LOCAL SCRIPT]
- Runs on the client (player's computer) only
- Has access to LocalPlayer and mouse
- Cannot modify server state directly
- Locations: StarterPlayer/StarterCharacterScripts and StarterPlayerScripts
- Examples: Movement.lua, ClientUI.lua

[MODULE SCRIPT]
- Reusable code shared between scripts
- Must be required() to use
- Can be used by both Server and Local scripts
- Location: ReplicatedStorage
- Examples: Constants.lua, Utilities.lua
